Dashboard

Welcome to the Promotions Management Dashboard. Here you can view statistics, manage users, and track promotions.

The Promotions Project is designed to help companies, promoters, and end users manage and participate in promotional campaigns efficiently. It provides a centralized platform to create, monitor, and analyze various promotional activities.

Key Features:

  • Admin Dashboard: Manage promoters, monitor campaign performance, and oversee all users.
  • Promoter Module: Create, edit, and track promotions — including start and end dates, descriptions, and target filters.
  • User Module: View and participate in available promotions based on your interests such as Sports, Music, Movies, Travel, and Reading.
  • Data Insights: Track how many users engage with each promotion and evaluate campaign reach.

Objective:

The main goal of this system is to streamline promotional campaign management, enhance user engagement, and provide data-driven insights to promoters and administrators.

Technology Stack:

  • Backend: Django (Python Framework)
  • Frontend: HTML, CSS, Bootstrap
  • Database: MySQL / SQLite
  • Template Engine: Django Templates

“A smarter way to connect promoters with users through targeted promotional campaigns.”
